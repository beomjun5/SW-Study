# SW Study

이 레포지토리는 다양한 프로그래밍 주제에 대한 스터디 자료를 정리한 공간입니다.  
각 폴더별로 주제를 나누어 학습 내용을 정리하고 실습 예제를 포함하고 있습니다.

## Directory Structure

| 디렉토리 | 설명 |
|-|-|
| [C](./C) | C 언어의 기초 문법, 자료형, 제어문, 함수 등 기초부터 다룹니다. |
| [OOP](./OOP) | Java를 중심으로 객체지향 프로그래밍(OOP)의 개념과 설계를 학습합니다. |

## 목표

- 기초부터 탄탄하게 개념을 정리
- 실습 중심으로 개념 내재화
- 팀원들과 함께 성장하는 스터디 환경 조성
